Project Overview

The Hangman Game is a classic word-guessing game developed using the MERN stack. It challenges players to guess a hidden word by suggesting letters.

Key Features

  • Random Word Generation: Each game starts with a randomly selected word from a database, givng a new challenge each time.
  • Session Management: Tracks player progress and scores across sessions.

Technologies Used

The game is built using the MERN stack (MongoDB, Express.js, React.js, Node.js).
