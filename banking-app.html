Project Overview

The Banking App is a user-friendly platform designed to streamline banking operations for admins, employees, and customers. It features dedicated dashboards for each user role, making it easy to manage accounts, handle transactions, and monitor activity securely.

Core Functionality

  • Admin Dashboard: Oversee the banking system, manage users, and transactions.
  • Employee Dashboard: Efficiently manage customer accounts and transactions.
  • Customer Dashboard: Access account details, transfer funds, and view transaction history.

Technologies Used

This app is built with React.js on the front-end and Node.js/Express.js on the back-end, with MongoDB for data storage.
