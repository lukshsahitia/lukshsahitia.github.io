Banking App

The Banking App is designed to handle various banking operations. It features three main interfaces: Admin, Employee, and Customer. Each interface is tailored to the needs of its respective users, ensuring secure and efficient management of banking tasks.

  • Admin Interface: Manage users, view transactions, and monitor system performance.
  • Employee Interface: Handle customer requests, process transactions, and manage accounts.
  • Customer Interface: Access account information, transfer funds, and view transaction history.

Mongo Accounts

This project focuses on managing user accounts using MongoDB, showcasing a practical application of NoSQL databases. It involves CRUD operations, user authentication, and secure data storage, providing a robust backend for web applications.

Hangman Game

The Hangman Game is a classic word-guessing game implemented using the MERN stack. The game features an intuitive user interface, random word generation, and session management to track high scores. It was built with agile development practices, ensuring a clean and maintainable codebase.

  • Random Word Generation: Words are chosen randomly from a database, making each game unique.
  • Session Management: Tracks player scores and game state across sessions.
  • Clean UI: A user-friendly interface that makes gameplay smooth and enjoyable.

Four in a Row

This project recreates the classic Four in a Row (Connect Four) game, with an emphasis on algorithmic game logic and user experience. The game features a dynamic interface, responsive design, and intuitive controls.
